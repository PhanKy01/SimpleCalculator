SimpleCalculator.java

Implemented:
_Infix to RPN.

_Supported operations:
__Parenthesis (only () is supported, [] and {} are not supported).
__Simple operation: +, -, *, /, ^
__Trigonometric functions (in radian): sin, cos, tan, cot
__Logarithmic functions: ln (natural logarithm), log (log base 10, or common log)
